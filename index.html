Installation

  1. Unzip the download and locate the .ccx file.
  2. Double-click the .ccx. The Creative Cloud desktop app opens and asks you to confirm.
  3. A warning about a plugin from outside Adobe's Marketplace is expected for self-distributed plugins — proceed.
  4. Restart Premiere Pro, then open Window > UXP Plugins > SFX Panel Pro.

To uninstall, use Manage plugins in the Creative Cloud desktop app.

FAQ

Does the panel collect any data?

No. Everything is stored locally. No account, no analytics, no network access.

Can placed SFX play back automatically?

No — Premiere's UXP API does not expose Program Monitor playback, so this is not possible in any UXP panel.

Can I assign keyboard shortcuts to SFX buttons?

No — Premiere does not allow key assignments for UXP panel commands. This is an Adobe platform limitation.

Which audio formats are supported?

mp3, wav, m4a, aif, aiff. (AIFF previews via the Source Monitor due to a Premiere webview limitation.)

(macOS) An SFX registers fine but fails to place with a "file could not be read" error.

This can happen with non-ASCII (e.g. Japanese) file names copied from Windows to macOS (Unicode NFC/NFD difference). Rename the files to plain alphanumeric names (e.g. sfx_001.wav) and re-register them.

I get a warning when double-clicking the .ccx.

That warning appears for any plugin installed from outside Adobe's Marketplace — it is safe to proceed.
